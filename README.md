# MPR POS — Prototipo de demostración

Prototipo estático del sistema POS para María Pestañas Rusas.

> ⚠️ DEMO — Sin datos reales. Solo para validación de UI/UX con el equipo.

## Flujo

Login → Abrir turno → POS → Cobrar → Recibo → Nueva venta

## Funciones incluidas

- Selección de cajera (Diana, Paola, Masha, Mario)
- Grid de productos con filtro por categoría y búsqueda
- Carrito con numpad (Cantidad / % Descuento / Precio libre)
- Pagos divididos entre múltiples métodos (Efectivo, Tarjeta, Transferencia, Mercado Pago, Clip)
- Cálculo de cambio automático
- Recibo imprimible
- Sidebar con hover-expand para navegación por módulos

## Tecnología

HTML + CSS + JS vanilla. Sin dependencias externas. GitHub Pages.
